# SkillUp - Student Skill Development Platform 🚀

## 📌 Description

SkillUp is a responsive student skill development platform website designed to help users explore and learn various skills like web development, communication, and digital marketing.

The project focuses on clean UI design, responsive layout, and basic frontend development using HTML and CSS.

---

## 🚀 Features

* 🏠 Home page with hero section and animation slider
* 📚 Courses page with multiple course cards
* 📩 Contact form with input validation
* 🎯 About section highlighting platform benefits
* 📱 Fully responsive design (mobile-friendly)
* 🎨 Clean UI with hover effects and animations

---

## 💻 Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Animations

---

## 📂 Project Structure

SkillUp/
│── index.html        (Home Page)
│── visit.html        (Courses Page)
│── contact.html      (Contact Page)

## ▶️ How to Run

1. Download or clone this repository
2. Open index.html in your browser
3. Navigate through Home, Visit, and Contact pages
